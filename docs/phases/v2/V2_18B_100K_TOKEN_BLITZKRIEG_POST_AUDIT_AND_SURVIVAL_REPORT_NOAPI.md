## Red Team Bulguları

- PNL_OPTIMISM_ILLUSION: V2_18A produced 100% win rate with zero loss, zero SL and zero sell failure. This is technically clean but market-realism unsafe. -> V2_18A_FIX1_PNL_REALISM_HARDENING_REQUIRED
- QUEUE_OR_CANDIDATE_BACKPRESSURE: Backpressure triggered while memory pressure was low. Root cause is queue/candidate-pool pressure, not RAM exhaustion. -> BACKPRESSURE_COMPONENT_TRACE_REQUIRED
- FP_FN_ROOT_CAUSE_REQUIRED: False positive=17, false negative=13. These 30 tokens require stage-level root-cause audit. -> FP_FN_STAGE_TRACE_REQUIRED
- HUMAN_TRIGGER_OVERLOAD: HUMAN_TRIGGER_REQUIRED=11241. Human gateway cannot manually review this volume. -> HUMAN_GATEWAY_TOP3_CAP_REQUIRED
- HUMAN_GATEWAY_LIMIT_NOT_ENFORCED: Human trigger was a classification state, not a strict final top-N execution gate. -> READONLY_HUMAN_INTERFACE_HARDBLOCK_REQUIRED
- SYNTHETIC_NETWORK_LATENCY_REQUIRED: Internal latency is excellent, but RPC/block/quote-TTL friction was not included in V2_18A original PnL. -> NETWORK_LATENCY_FRICTION_MODEL_REQUIRED
- ADAPTIVE_THREAT_EVOLUTION_REQUIRED: Static threat model cannot cover future tax/router/proxy/evasion patterns without learning/replay cycles. -> THREAT_EVOLUTION_REPLAY_REQUIRED
