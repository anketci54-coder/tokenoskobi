  -> RISK ENGINE

## 02 DATA FLOW

RAW
  -> STAGING
  -> VALIDATED
  -> CANONICAL
  -> READMODEL
  -> PANEL

## 03 RISK FLOW

  -> SELLABILITY CHECK
  -> LIQUIDITY CHECK
  -> CONTRACT / DEPLOYER / HOLDER RISK
  -> HONEYPOT / RWD / TRAP CHECK

  -> ALLOW / BLOCK / WAIT / REVIEW
